# Final Full Project

This ZIP contains:

- `frontend/` → Angular CLI 21 frontend
- `Backend/` → Express.js + MongoDB backend

## Run backend

```bash
cd Backend
npm install
copy .env.example .env
npm start
```

## Run frontend

```bash
cd frontend
npm install
npm start
```

## URLs

- Frontend: `http://localhost:4200`
- Backend: `http://localhost:5600`

## Main features

- User registration
- User login with JWT
- Protected home page
- Product add, view, update, delete
- MongoDB database integration
