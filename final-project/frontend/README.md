# Angular CLI 21 + Express.js + MongoDB Project

## Frontend setup

```bash
cd frontend
npm install
npm start
```

Frontend runs on:

```bash
http://localhost:4200
```

## Backend setup

```bash
cd Backend
npm install
npm start
```

Backend runs on:

```bash
http://localhost:5600
```

## Features

- Angular CLI 21 standalone frontend
- Register and login
- JWT token stored in localStorage
- Protected home page
- Product CRUD
- Express.js + MongoDB backend
